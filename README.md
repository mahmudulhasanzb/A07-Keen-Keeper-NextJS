# Keen Keeper

A personalized relationship management dashboard designed to help you stay connected with the friends who matter most. 

## Features
- **Friend Management:** Keep track of contact frequency, tags, status, and set personal relationship goals.
- **Interactive Check-ins & Timeline:** Log quick interactions (calls, texts, videos) and view a rich history of connections.
- **Analytics Dashboard:** Visualize your interactions over time using a dynamic pie chart to better understand your communication habits.

## Technologies Used
- **Next.js & App Router:** For robust structure and fast server/client routing.
- **Tailwind CSS & DaisyUI:** Used for quick styling, responsive layout, and beautiful UI components.
- **Recharts:** Providing the Friendship Analytics data visualization.
- **React Hot Toast:** Giving sleek popup feedback upon interaction logging.
